# crud-node.js
# Cuando clonen el proyecto u neseciten volver a instalar los modulos de JS simplemente ejecuten "npm i"

# npm i dotenv: este comando es para leer variables de entorno, es decir luego de ejecutarlo puedes leer variables de entorno.

# npm i -D nodemon: La "-D" hace referencia a que va a instalar un paquete de desarrollo, este paquete de desarrollo se llama "nodemon"
# y es para que al actulizar los cambios estos se vean reflejados en su dominio y no tener que finalizar la ejecucuion con contrl+c y 
# volver actulizarejecutar para ver los cambios.

# npm install express: para instalar express.

# Para ejecutar el proyecto en modo desarrollador ejecute "npm run dev"

# npm i ejs: para instalar el motor de vistas.
